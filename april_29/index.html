Front End

HTML (hyper text markup language) is the basic of every website. We use HTML in order to delineate our site and create its basic structure. HTML uses tags that create the structure of our site. In the example on the right, we have a phone screen with basic markup. We use HTML to create the raw input boxes for a basic signin form. We have fields for username and password; also, there is a submit button. Right now we have only HTML and it looks like that:

<div class='submit-form'>

- we declare a div tag and wrap the whole submit form within.

<input type="text" placeholder=username>

- With HTML we create an input box for the username

<input type="text" placeholder=password>

- another input box for the password

<input type="submit">

- finally submit button

<div class='submit-form'>
