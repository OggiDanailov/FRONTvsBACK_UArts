Front End

When we open any browser (Chrome, Safari, Firefox etc.) we always see some sort of information which usually is quite visual - there are images, texts, inputs where we can type. This part is referred as Front end, or also Client side.

Usually there are three languages that are commonly used in the Front end development - HTML (hyper text markup language), CSS (cascading style sheets) and JavaScript (the only Front end programming language). Front end developers use these three languages (not necessary to use all of them always) in order to create the markup of each page, the styling of each element and, if necessary, interactive elements with which users can interact. Further down we will explore what is the role of each if these languages and what exactly they do

HTML

HTML is a programming tool that determines the visual layout of the content on every web page. We call it a “markup language” because of the way it shapes the overall design and layout, from titles and headings to blocks of text. When we talk about HTML, we refer to “tags,” which are the hidden keywords and phrases that — when deployed on a web page — create the format, or “skeleton,” of a web page.

CSS

We use CSS to style the already created HTML elements. With the help of CSS we can do various things - we can move the HTML elements on our site, we can change the size of them, we can give them colors, shadows and many other things. CSS can be very artistic and at the same time complex.

Below we will use CSS in order to style our HTML elements. Click on any of them to see how they will affect our site

div {height: 100px}


p {margin-left: 40px; box-shadow: 4px 4px 4px grey}


h3 {color: red; font-size: 30px}
